HTML Tags - Audio - Video - Canvas

Audio Tag

Here is the default audo tag. As you can see, the default tag needs "controls" attribute to be meaningful. Without it, the user cannot interact with audio on a web page. 

"<audio controls></audio>"

Here is some audio from the first chapter of the audiobook, Maze Runner by James Dashner. Notice that the default controls allow access to play/pause, volume control, progress and progress scan, and track length.



Setting the "autoplay" to "On" or "Off" will persist regardless of browser session. If "autoplay" is set to "On", when you leave the page/session and come back, the audio file will autoplay.

Video Tag





The video tag allos for video to be embedded or referenced on a web page. You can control the video much like the audio, using default controls. Using DOM (Document Object Model), you can make your own conrols, like the buttons around the video window. Using the "Play/Pause" button, you can play or pause the video playback. Using the "Big", "Small", or "Normal" buttons, you can resize the video. Using the "Controls" toggle button, you can remove or add default controls to the video window. Note, using the "Controls" button will cause the video to reload in order to reflect the change.

Canvas Tag

The canvas tag can do many things, but it's basically designed to handle:

  • Drawing text
  • Graphics
  • Animations
  • Interactions (like mouse clicks, button clicks, key clicks, etc.)
  • HTML based games

In our case, we'll use canvas to draw a clock.
