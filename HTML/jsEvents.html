Using the box below, perform different events that will change the color of the box.

  • Click the button to cycle all the colors
  • Click the box area to change the color to blue
  • Mouse over the box to change the color to yellow
  • Strike any key on the keyboard to change the color red (must click inside box first)
  • Once the cursor is in the box, moving it out of the box will change the color green
  • View the counter below the box to see how many changes you've performed
