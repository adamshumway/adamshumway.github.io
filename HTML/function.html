Javascript Function(s), Arrays, and Loops

Simple Function:

This is a simple function that multiplies two numbers together. 

            
                function myFunction(p1, p2) {
                    return p1 * p2; // The function returns the product of p1 and p2
                }
                document.getElementById("function").innerHTML = myFunction(4, 6);
This function has two parameters, p1 and p2. The parameters accept input for the function and then use that input to do work inside the function. In our case, we're just multiplying the paramters together. As you can see, we are calling the function as part of the document innerHTML call and passing the numbers 4 and 6. For Loop example:

        
            for (i = 0; i < 5; i++) {
                text += "The number is " + i + "<br>";
            }

In this loop, we iterate from 0 to 10 and output the list of items in the Array, Car Colors.

Example of For In Loop:

        
            var person = {fname:"John", lname:"Doe", age:25}; 

                var text = "";
                var x;
                for (x in person) {
                    text += person[x];
            }

As you can see, the "For In Loop" has the same basic loop idea, you just don't have to specify the loop iteration. This is mostly because a "For In Loop" will iterate through an associative array, otherwise known as an array object. An Object array will have key value pairs as seen in the above "person" example. We'll run the similar data as before, but as a key value pair object array instead.

While Loop:

The while loop incorporates an important element in javascript, the condition. A condition is an evaluation that more or less evaluatest to true or false. For example: 

            
                var i = 5;
                while (i > 3)
                {
                    i--;
                }
In this example, we set the variable "i" to the number 5. in the wile loop, as long as i (5) is greater than 3, the code inside the braces {} will execute. In this case, we are decrementing the variable with each iteration. Once i = 3, the condition of the while loop will be met and we will exit the loop. To demonstrate this, when the button is clicked, a while loop will execute and a separate counter will count the iterations of the loop. Using our example, we'll set a variable to 5 and iterate until 5 is no longer greater than 3. It should be two iterations.
